# Python Practice Projects

This repository contains Python practice programs created while learning programming fundamentals, problem-solving, and logical thinking.

The projects cover various topics such as:

- Basic Input/Output
- Conditional Statements
- Loops
- Functions
- Lists and Arrays
- Mathematical Calculations
- Searching Algorithms
- Pattern Printing
- Number Theory

---

## Projects

| Project Name | Description |
|-------------|-------------|
| Area_Calculator.py | Calculates the area of different geometric shapes. |
| Half_Ascending_and_Half_Descending.py | Arranges a list with the first half in ascending order and the second half in descending order. |
| To_find_target.py | Finds a target element in a list. |
| count_even_odd_in_list.py | Counts even and odd numbers in a list. |
| countries_war.py | Practice project using country-related data and conditions. |
| even_of_infinity_range.py | Prints or processes even numbers in a given range. |
| largest_and_smallest.py | Finds the largest and smallest elements. |
| largest_no_in_list.py | Finds the largest number in a list. |
| odd_infinity.py | Prints or processes odd numbers in a given range. |
| prime_no_infinity_to_infinity.py | Checks and generates prime numbers. |
| root_of_infinity.py | Calculates square roots for a range of numbers. |
| sum_of_list.py | Calculates the sum of list elements. |
